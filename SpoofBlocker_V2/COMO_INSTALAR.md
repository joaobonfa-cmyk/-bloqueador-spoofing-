# Como Instalar o App sem o Android Studio

Para instalar o aplicativo no seu telefone sem precisar configurar um ambiente de desenvolvimento, siga estes passos simples usando o GitHub:

## Passo 1: Criar um Repositório no GitHub
1. Acesse [github.com](https://github.com) e faça login (ou crie uma conta gratuita).
2. Clique no botão **"+"** no canto superior direito e selecione **"New repository"**.
3. Dê um nome ao repositório (ex: `meu-bloqueador-spoofing`).
4. Selecione **"Public"** e clique em **"Create repository"**.

## Passo 2: Enviar os Arquivos
1. No seu novo repositório, clique em **"uploading an existing file"**.
2. Arraste todos os arquivos da pasta que eu te enviei para lá.
3. Clique em **"Commit changes"** no final da página.

## Passo 3: Gerar o APK (Automático)
1. Clique na aba **"Actions"** no topo da página do seu repositório.
2. Você verá um item chamado **"Android CI"**. O GitHub começará a construir o seu app automaticamente.
3. Aguarde alguns minutos (cerca de 2-4 min) até que apareça um ícone de check verde ✅.
4. Clique no nome do build que terminou.
5. Role a página até a seção **"Artifacts"** e clique em **"app-debug"** para baixar o arquivo ZIP.

## Passo 4: Instalar no Celular
1. Extraia o arquivo ZIP baixado; você encontrará o arquivo `app-debug.apk`.
2. Envie esse arquivo para o seu celular (via WhatsApp, Google Drive ou cabo USB).
3. No celular, abra o arquivo APK. 
   - *Nota: O Android pedirá para permitir a "Instalação de fontes desconhecidas". Pode permitir, pois este é o app que acabamos de criar.*
4. Siga as instruções na tela para concluir a instalação.

## Passo 5: Configurar o App
1. Abra o app **"Bloqueador de Spoofing"**.
2. Clique no botão para permitir que ele seja o seu **"Filtrador de Chamadas"**.
3. Digite o prefixo que deseja bloquear (ex: `2199257`).
4. Clique em **Salvar**.
